# Ejercicio Práctico: Gestión de Stock y Notificaciones

**Objetivo:** Desarrollar una pantalla de gestión de inventario simple utilizando Android Jetpack Compose y Navigation.

## 1. Requerimientos de Funcionalidad
- El sistema debe manejar una variable de estado llamada `cantidadProducto` que inicie en **5**.
- Debe haber un botón con el texto **"Vender Unidad"**.
- Al pulsar el botón, el valor de `cantidadProducto` debe **disminuir en 1**.

## 2. Lógica de Navegación y Alertas
- Si la cantidad llega a **0**, la aplicación debe navegar automáticamente a una pantalla de **"Agotado"**.
- La pantalla de "Agotado" debe mostrar el mensaje: "No quedan existencias disponibles".

## 3. Restricciones Técnicas
- Separe la navegación en un archivo de rutas independiente.


## 4. Criterios de Evaluación
- Correcta implementación del ciclo de vida del estado.
- Capacidad de respuesta ante cambios solicitados en tiempo real.
